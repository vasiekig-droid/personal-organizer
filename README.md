# Мой нежный органайзер v2.2

## Что нового

- Google Calendar теперь выгружается в календарь внутри органайзера как расписание занятости.
- События Google не превращаются в задачи и не меняются в Google Calendar.
- При подключении к интернету приложение пробует автоматически обновить занятость из Google, если Google уже был подключён и включена настройка автообновления.
- В календаре органайзера дела и события Google видны по дням вместе.
- События Google можно отмечать галочкой и вычёркивать локально.
- Обновлена красивая иконка приложения.

## Как обновить на GitHub

Загрузить в корень репозитория personal-organizer только файлы из этой папки:

- index.html
- manifest.json
- sw.js
- icon.svg
- README.md

После загрузки открыть:

https://vasiekig-droid.github.io/personal-organizer/?v=22

Если кэш держит старую версию:

https://vasiekig-droid.github.io/personal-organizer/?v=22-2

## Важно

GitHub Sync и Google Calendar требуют интернет. Офлайн работают локальные задачи, заметки, финансы и сохранённая последняя выгрузка Google-событий.
