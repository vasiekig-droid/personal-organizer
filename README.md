# Мой нежный органайзер v2.3

Версия с устойчивыми настройками: GitHub Sync, Google Calendar Client ID и параметры синхронизации хранятся отдельно от данных приложения и не должны слетать при обычных обновлениях.

## Как обновить

Загрузите в корень репозитория `personal-organizer` файлы из папки `orgv23`:

- `index.html`
- `manifest.json`
- `sw.js`
- `icon.svg`
- `README.md`

Открыть: `https://vasiekig-droid.github.io/personal-organizer/?v=23`

## Важно про токены

GitHub token хранится только локально на устройстве и не отправляется в `organizer.json`. Это безопаснее. Чтобы не потерять его при переустановке приложения или очистке данных сайта, используйте в настройках кнопку **Скачать настройки** и храните файл только у себя.
