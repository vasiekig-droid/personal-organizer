# Мой нежный органайзер v2.7 — быстрые финансы

Что нового:
- добавлен режим быстрого ввода трат для телефона;
- добавлен PWA shortcut “Записать трату”;
- можно открыть быстрый финансовый режим ссылкой: `?quick=finance`;
- быстрый ввод одной строкой: `850 продукты молоко хлеб`;
- обычные настройки GitHub/Google сохранены.

Обновление на GitHub Pages:
1. Загрузить в корень репозитория файлы из папки `orgv28`.
2. Открыть: `https://vasiekig-droid.github.io/personal-organizer/?v=27`.
3. Для быстрого финансового ярлыка: `https://vasiekig-droid.github.io/personal-organizer/?v=27&quick=finance`.

Важно: GitHub token не хранить в репозитории и не отправлять в чат.
