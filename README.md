## Обновление на GitHub

Загрузите в корень репозитория personal-organizer файлы из папки orgv21:

- index.html
- manifest.json
- sw.js
- icon.svg
- README.md

Открывайте приложение так:

https://vasiekig-droid.github.io/personal-organizer/?v=21

Если браузер показывает старую версию, откройте:

https://vasiekig-droid.github.io/personal-organizer/?v=21-2

Важно: organizer.json не удалять и не перезаписывать руками.
